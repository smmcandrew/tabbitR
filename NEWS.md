# tabbitR 0.1.0

* Initial public release.
* Provides `tabbit_excel()` for producing weighted and unweighted cross-tabulation
  tables exported directly to Excel.
* Supports multiple outcome and breakdown variables, transparent handling of
  missing values, optional row/column percentages, and light Excel formatting.

# tabbitR 0.1.3
* Removed message side effects during function execution.
* Examples now write output to temporary files.
